## Project Overview

**Important**: This directory appears to be the output of `jekyll build` (typically the `_site` directory). The actual Jekyll source files with `_config.yml`, `Gemfile`, `_posts/`, `_layouts/`, etc. are not present in this directory.

## Architecture Notes

- **Theme**: Uses the Minimal Mistakes Jekyll theme
- **Responsive Design**: Mobile-friendly with viewport meta tags
- **Legacy Browser Support**: Includes HTML5 shiv and Modernizr for IE compatibility
- **JavaScript Libraries**: jQuery 1.9.1, Magnific Popup (lightbox), FitVids (responsive videos)
- **Icons**: FontAwesome for social media and UI icons
- **Feed**: RSS/Atom feed available at [feed.xml](feed.xml)
- **Sitemap**: XML sitemap at [sitemap.xml](sitemap.xml)
